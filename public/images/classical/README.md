# Classical Images Directory

Place your classical Greek statue or philosopher images in this directory.

## Required Images

1. **statue-login.jpg** - For the login screen
   - Front-facing composition
   - Classical Greek statue or philosopher
   - Engraving style, heavy halftone, monochrome
   - Dot-matrix aesthetic, brutalist lighting

2. **philosopher-welcome.jpg** - For onboarding screen 1 (Welcome)
   - Same aesthetic requirements as above

3. **statue-ascension.jpg** - For onboarding screen 4 (Begin Your Ascension)
   - Same aesthetic requirements as above

## Image Specifications

- **Format**: JPG or PNG
- **Style**: Engraving style, heavy halftone, monochrome
- **Aesthetic**: Dot-matrix, brutalist lighting
- **Composition**: Front-facing
- **Recommended Size**: 300x400px or similar portrait orientation
- **Color**: Monochrome (will be converted to grayscale via CSS filters)

## CSS Effects Applied

The images will automatically receive:
- Heavy halftone overlay
- Dot-matrix texture
- Engraving-style filters (contrast, brightness, grayscale)
- Brutalist lighting effects
- PS1-era pixel grain

All effects are applied via CSS, so you can use any monochrome classical image and it will be transformed to match the Ratio aesthetic.
